# I/O session log file format for Vim

## Example

    > insert 20
    < Element inserted
    > delete 20
    < Element deleted
    > delete 25
    > Element not found

## Installation

Clone the repository into ~/.vim/bundle and load it with [Pathogen][], or use [Vundle][].

  [pathogen]: https://github.com/tpope/vim-pathogen
  [vundle]: https://github.com/gmarik/vundle
